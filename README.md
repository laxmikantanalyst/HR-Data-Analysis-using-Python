## 📌 Project Overview

This project analyzes 100,000+ employee records using Python to uncover meaningful insights related to workforce distribution, employee attrition, salary trends, performance, and hiring patterns.

### 🔧 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

### 📈 Key Insights

- Overall Attrition Rate: **13.20%**
- IT Department has the highest employee count.
- Most employees work in **On-site** mode.
- High Performers (4 & 5 Star) make up **55.2%** of the workforce.
- Salary increases with employee experience.
- HR Department has the highest attrition rate.
